# ZDC - Kitchen Generator

Complete kitchen scene/layout generation system for Blender 5.0+.

## Status

**Stub** — register/unregister scaffolding only. No functionality implemented yet.

## Planned Features

See the ProKitchen specification documents in `docs/` for the full design:

- Room construction (walls, floor, ceiling) from dimensions
- Cabinet layout with automatic spacing and filler logic
- Appliance placement with cutout generation
- Countertop generation with edge profiles
- Material assignment from finish libraries
- Integration with cabinet-generator for parametric cabinet models
