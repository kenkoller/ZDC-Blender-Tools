# Design Documents

Specification and design documents for ZDC Blender Tools.

## ProKitchen Specs

Detailed specifications for the unified kitchen generation system:

- `ProKitchen_Unified_Kitchen_System.md` — Architecture overview, component hierarchy, room flow
- `ProKitchen_Kitchen_Components_Spec.md` — Individual component specifications (cabinets, appliances, countertops)
- `ProKitchen_Material_System_Spec.md` — Material system design, finish libraries, texture handling

These specs inform the development of the kitchen-generator and cabinet-generator addons.
