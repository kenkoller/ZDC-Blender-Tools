# Cabinet Generator

Parametric cabinet generation addon for Blender 5.0+.

## Features (Planned)
- Real-time parameter adjustment via Geometry Nodes
- Multiple cabinet styles (base, upper, vanity, pantry)
- Configurable doors, drawers, and shelving
- Hardware placement system
- Material assignment

## Installation
1. Download or clone this repository
2. In Blender: Edit > Preferences > Add-ons > Install
3. Select the cabinet-generator folder
4. Enable "Cabinet Generator" addon

## Usage
1. Open sidebar (N key) in 3D Viewport
2. Find "Cabinet Generator" tab
3. Adjust parameters and click Generate

## Requirements
- Blender 5.0 or newer
